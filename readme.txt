This app has a model which has three fields
    1 :username
    2 :email
    3 :password

there is a form where users can create account , these three fields want to be unique and some more validation are also there
for all users if a user enters any value which already exists then it will show an
error in form that this field value already exists.

For this in this project we are using ajax (fetch)

Additional Validation :

    1.Username
        first charecter want to be capital
        minimum length 5 characters
        unique value
    
    2.Email
        email want to contain ' @ ' and '.' in the value
        unique value
    
    3.Phone
        length 10 digits
        unique value
